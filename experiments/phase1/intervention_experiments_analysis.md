# 层级干预实验完整分析

> 模型：Qwen3-1.7B (28层, d_model=2048) | 数据集：HellaSwag (4选1常识推理) | 随机基线：25%

---

## 一、背景：从检测到干预

### 1.1 检测阶段的核心发现

在方向一的检测阶段（`main.py` + `analyze_wauroc.py`），我们对 Qwen3-1.7B 逐层评估了 AUROC 和 wAUROC：

| 数据集 | 准确率 | AUROC | wAUROC | wAUROC 最优层 |
|--------|--------|-------|--------|------------|
| HellaSwag | 54% | 0.694 | **0.809** | **L11** |

HellaSwag 的 wAUROC 达到 0.809，意味着在模型"声称知道"的样本中，L11 残差流的内部信号能以 80.9% 的准确率区分正确和错误预测。

**这引出了一个因果问题**：L11 是检测最优层，但它在因果上真的最重要吗？如果我们在 L11 注入噪声，会破坏模型的正确预测吗？

### 1.2 干预实验的理论框架

**残差流视角**：Transformer 的每一层通过残差连接向主干添加信息：

$$h_{\ell+1} = h_\ell + \text{Attn}_\ell(h_\ell) + \text{MLP}_\ell(h_\ell)$$

我们通过 TransformerLens 的 hook 机制，在特定层的残差流输出（`blocks.{i}.hook_resid_post`）处注入扰动。

**两种扰动类型**：

| 类型 | 公式 | 含义 |
|------|------|------|
| 噪声注入 | $h \leftarrow h + \sigma \cdot \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, I)$ | 在 d_model 维空间添加各向同性高斯噪声 |
| 缩放 | $h \leftarrow \alpha \cdot h$ | 逐元素缩放残差流幅度 |

**实验逻辑**：
- 如果某层的残差流承载了模型正确预测所需的关键信息，扰动应导致准确率下降
- 不同层对同一扰动的敏感度差异，揭示了信息在不同层级的因果重要性分布

## 二、初始干预实验（n=100）

### 2.1 实验设计

- **样本**：HellaSwag n=100，基线准确率 51%（51/100）
- **干预位点**：
  - L0（嵌入层输出，`blocks.0.hook_resid_pre`）
  - L11（检测最优层，`blocks.10.hook_resid_post`）
  - L14（中间对照层，`blocks.13.hook_resid_post`）
- **噪声**：$\sigma \in \{0, 0.01, 0.05, 0.1, 0.5, 1.0, 2.0, 5.0\}$
- **缩放**：$\alpha \in \{0, 0.1, 0.3, 0.5, 0.7, 0.9, 1.0\}$

### 2.2 噪声注入结果

| σ | L11 (检测最优) | L0 (嵌入层) | L14 (对照) |
|----|--------------|-----------|----------|
| 0.0 | 0.51 | 0.51 | 0.51 |
| 0.01 | 0.51 | 0.50 | 0.51 |
| **0.05** | 0.50 | **0.13** | 0.50 |
| **0.1** | 0.52 | **0.00** | 0.50 |
| 0.5 | 0.42 | 0.00 | 0.45 |
| 1.0 | 0.40 | 0.00 | 0.42 |
| 2.0 | 0.14 | 0.00 | 0.35 |
| 5.0 | 0.01 | 0.00 | 0.02 |

### 2.3 缩放结果

| α | L11 (检测最优) | L0 (嵌入层) | L14 (对照) |
|----|--------------|-----------|----------|
| 0.0 | 0.00 | 0.00 | 0.00 |
| 0.1 | 0.00 | 0.46 | 0.02 |
| 0.3 | 0.05 | 0.47 | 0.02 |
| 0.5 | 0.33 | 0.48 | 0.49 |
| **0.7** | 0.47 | 0.50 | **0.56** |
| 0.9 | 0.49 | 0.51 | 0.51 |
| 1.0 | 0.51 | 0.51 | 0.51 |

### 2.4 核心发现（推翻初始假设）

1. **"检测最优 ≠ 因果最重要"**：L11 wAUROC 最优（0.809），但 L11 对噪声最鲁棒（σ=1.0 仍保持 0.40 准确率）。L0 AUROC 一般，却对噪声极度脆弱（σ=0.05 → 准确率从 0.51 崩塌至 0.13）。

2. **噪声与缩放的不对称性**：L0 对噪声敏感但对缩放鲁棒（α=0.1 → 仍保持 0.46）；L11 相反——对缩放敏感（α=0.3 → 0.05）但噪声鲁棒（σ=1.0 → 0.40）。

3. **L14 缩放提升效应**：α=0.7 时准确率从 0.51 提升至 0.56（+5pp），暗示该层残差流中可能存在"有害子空间"，适度抑制改善了预测。

4. **解释**：嵌入层噪声破坏 token identity，造成级联失效。中间层表示信号在方向上分布均匀（各向同性噪声不敏感），但幅值编码关键区分信息（缩放敏感）。L14 可能累积了偏向错误答案表面模式的特征。

## 三、方案 A：L14 缩放验证

### 3.1 研究问题

L14 α=0.7 时 +5pp（n=100）的提升效应是否可复现？是否 L14 特有？是否统计显著？

### 3.2 实验设计

- 样本：n=300（增大 3 倍以提升统计效力）
- 层：L11, L12, L13, L14, L15, L16（6 层，覆盖甜区 + 对照组）
- α 范围：0.55 ~ 0.85，step=0.05（7 个级别，细粒度扫描）
- 统计：McNemar 配对检验（二项检验，双侧），比较每样本正确/错误的配对变化
- 共 6 × 7 = 42 个实验条件

### 3.3 结果

基线准确率：52.33%（157/300）

| 层 | 最优 α | 最优 Acc | Δ | McNemar p | 显著? |
|---|--------|---------|---|-----------|------|
| L11 | 0.85 | 50.00% | −2.3pp | 0.26 | 否 |
| L12 | 0.85 | 51.67% | −0.7pp | 0.84 | 否 |
| L13 | 0.80 | 53.67% | +1.3pp | 0.56 | 否 |
| **L14** | **0.65** | **56.33%** | **+4.0pp** | 0.12 | 否 |
| L14 | 0.75 | 56.33% | +4.0pp | 0.08 | 否 |
| L15 | 0.55 | 56.00% | +3.7pp | 0.24 | 否 |
| L16 | 0.85 | 50.00% | −2.3pp | 0.07 | 否 |

### 3.4 结论

1. **L14 效应成功复现**：n=100 时 51%→56%（+5pp），n=300 时 52.3%→56.3%（+4pp），效应量一致，非实验噪声
2. **非 L14 独有**：L13（+1.3pp）和 L15（+3.7pp）也出现提升——这是一个跨越 L13-L15 的"衰减甜区"
3. **n=300 未达统计显著**：McNemar p=0.08-0.12。效应量约 +4pp，估算需要 n≈500-800 才能 p<0.05
4. **甜区分布**：L13-L15, α∈[0.55, 0.80]。控制层 L11 和 L16 一致退化（多组 p<0.001）

**关键推论**：效应真实但微弱。这意味着衰减对大多数样本帮助有限或略微有害，但对特定子集有较大帮助。后续需要分析哪些样本从衰减中受益，以理解甜区的机制。

## 四、方案 B：L11 方向性噪声

### 4.1 研究问题

L11 对噪声鲁棒，是因为所有方向都同等重要（信息分布式），还是因为噪声能量分散到了不重要方向（信息集中在低维子空间）？

**假设**：如果 L11 的检测信号集中在正确答案的 unembedding 方向附近，那么沿该方向的噪声应比各向同性噪声造成更大退化。

### 4.2 实验设计：方向性噪声

给定 unembedding 矩阵 $W_U \in \mathbb{R}^{d_{model} \times V}$，token $t$ 的 unembedding 向量为 $u_t = W_U[:, t]$。定义单位方向向量 $d = u_t / \|u_t\|$。

四种噪声模式：

| 模式 | 公式 | 含义 |
|------|------|------|
| Isotropic | $\varepsilon \sim \mathcal{N}(0, I)$ | 全方向等权，基准对照 |
| Parallel (correct) | $\langle \varepsilon, d_{correct} \rangle \cdot d_{correct}$ | 仅在正确答案方向加噪（1 维） |
| Parallel (predicted) | $\langle \varepsilon, d_{pred} \rangle \cdot d_{pred}$ | 仅在模型预测方向加噪（1 维） |
| Orthogonal (correct) | $\varepsilon - \langle \varepsilon, d_{correct} \rangle \cdot d_{correct}$ | 在所有方向加噪，但跳过正确方向（2047 维） |

**关键对比逻辑**：
- 若 Parallel >> Isotropic 在相同 σ 下退化更严重 → 该方向承载关键因果信息
- 若 Orthogonal ≈ Isotropic → 保留该方向几乎无帮助，信息不是低维的
- 若 Parallel 几乎不退化 → 该方向基本无因果重要性

### 4.3 结果

基线准确率：54.00%（108/200），σ=1.0 时：

| 噪声模式 | 准确率 | Δ | 噪声维度 |
|---------|--------|----|---------|
| **Isotropic** | 33.0% | −21.0pp | 2048 |
| **Parallel (correct)** | **54.0%** | **0.0pp** | 1 |
| Parallel (predicted) | 53.5% | −0.5pp | 1 |
| Orthogonal (correct) | 36.5% | −17.5pp | 2047 |

**即便 σ=5.0**：Parallel (correct) 仍保持 53.5%，Parallel (predicted) 保持 52.0%。

### 4.4 结论

1. **单一 unembedding 方向无因果重要性**：沿正确答案或预测方向注入噪声，即便 σ=5.0 也几乎不影响准确率
2. **正交方向噪声逼近各向同性**：2047 维噪声与全 2048 维噪声退化幅度接近（−17.5pp vs −21.0pp）
3. **"低维子空间"假说被明确否定**：L11 的检测信号高度分布式地编码在整个 d_model 空间中，不存在特权方向
4. **与 Plan A 的呼应**：L11 的分布式表征解释了为什么缩放比噪声更有效地扰动 L11——缩放同时影响所有维度，而各向同性噪声的能量被分散到无关方向

**方法论洞察**：方向性噪声是一个有效的因果探测工具。通过对比"所有方向"、"单个方向"、"除单个方向外的所有方向"，可以区分分布式编码 vs 低维编码。

## 五、方案 C：L0 噪声崖壁精确定位

### 5.1 研究问题

初始实验（n=100）发现 L0 噪声 σ=0.05 时准确率从 0.51 崩塌至 0.13，σ=0.10 时归零。这暗示可能存在尖锐的相变阈值。本实验以更高分辨率精确定位崩塌位置。

### 5.2 实验设计

- 样本：n=200
- 干预位点：L0（`blocks.0.hook_resid_pre`，嵌入层输出）
- σ 范围：0 ~ 0.10，步长 0.005（19 个级别，比 n=100 实验密 4 倍）
- 分析指标：半衰 σ（准确率降至基线一半时的 σ 值）、最陡降落点

### 5.3 结果

基线准确率：54.00%（108/200）

| σ 区间 | 现象 | 准确率范围 |
|--------|------|-----------|
| 0.005 | 微弱扰动，甚至略高于基线 | 54.5% |
| 0.01 - 0.025 | 缓慢退化，相对稳定 | 52.0% → 45.5% |
| 0.03 - 0.045 | 加速下降 | 35.5% → 26.0% |
| **0.05** | **崩塌点** | **7.5%** |
| 0.06 - 0.07 | 接近失效 | 0.5% |
| ≥ 0.08 | 完全失效 | 0.0% |

**半衰 σ ≈ 0.04**（准确率降至 27.0%），最陡降落 σ=0.045→0.050（Δ=18.5pp）。

### 5.4 结论

1. **没有尖锐的一阶相变**：初始实验的粗粒度采样（σ=0.01→0.05）让退化看起来像跳变，实际上是一个加速但连续的退化过程
2. **半衰阈值 ~0.04**：这是嵌入层对噪声的"容忍度"量化指标——σ 超过此值，token identity 信息被破坏到无法恢复的程度
3. **σ=0.005 的微弱提升（+0.5pp）**：可能是随机波动，也可能是微小噪声带来的隐式正则化——不排除真实存在但效应极小
4. **与 Plan B 对比**：L0 噪声 σ=0.05 就崩塌（7.5%），L11 噪声 σ=5.0 仍有 53.5%（平行方向）。嵌入层的因果脆弱性比中间层高出两个数量级。

## 六、综合总结

### 6.1 核心发现矩阵

| 发现 | 证据来源 | 确定性 |
|------|---------|--------|
| "检测最优 ≠ 因果最重要" | 初始实验 + A + B | ★★★★★ |
| L0 是因果瓶颈，σ≈0.04 即崩塌 | 初始实验 + C | ★★★★★ |
| L11 信号高度分布式，无特权子空间 | B | ★★★★★ |
| L13-L15 衰减甜区 (+4pp)，效应真实但弱 | 初始实验 + A | ★★★★☆ |
| L0 不是尖锐相变，是加速连续退化 | C | ★★★★☆ |
| L11 对缩放比对噪声更敏感 | 初始实验 | ★★★★☆ |

### 6.2 理论解释框架

```
L0 (嵌入层)          L11 (检测最优)        L14 (过渡区)
    │                     │                    │
    │  token identity     │  知识区分信号       │  有害子空间积累
    │  对噪声极度脆弱      │  分布式编码         │  适度抑制有利
    │  对缩放鲁棒          │  对缩放敏感         │
    │                     │                    │
    ▼                     ▼                    ▼
 级联失效              信号逐层退化          噪声鲁棒但可优化
```

**统一解释**：
- L0 的脆弱性来自其表示基础性——token identity 信息一旦被噪声破坏，所有后续层的计算都基于错误输入
- L11 的分布式表征使其对任何单一方向扰动不敏感——信息被冗余编码，需要同时扰动多数维度才能破坏
- L13-L15 的甜区效应暗示这些层可能过度拟合了训练数据中的表面模式（如错误答案的语词偏好），适度抑制反而纠正了偏差
- 缩放比噪声更有效扰动深层，因为缩放同时作用于所有维度而不分散能量

### 6.5 方法论教训

1. **先粗后细**：先用粗粒度 n=100 扫描确认效应存在，再用大样本+细粒度验证，避免浪费计算
2. **逐样本配对统计**：McNemar 检验利用配对设计，比独立样本检验更敏感；保存 `per_sample_correct` 是必要的
3. **控制层 > 2 个**：Plan A 从 3 层扩展到 6 层，发现甜区跨越 L13-L15——仅测 L14 会误判为单层效应
4. **方向性噪声是因果探针**：通过对比 1 维 vs d-1 维噪声，无需低维假设即可探测表征的分布特性
5. **JSON 序列化陷阱**：numpy bool/float 不能直接序列化，需显式转 Python 原生类型
