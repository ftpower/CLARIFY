# I1：对比方向引导干预 — 原理与实现

## 1. 问题背景

Phase 2 的 **检测路线** 发现：知识筛选后，max_p AUROC 可达 0.87（D2 联合 JS 散度可达 0.936），即我们有能力在推理时判断模型"是否会答错"。

**干预路线** 的目标是：在检测到风险后，修改模型的内部状态以提升准确率。但此前所有干预方案（各向同性噪声注入、VIB 瓶颈、DoLa 对比解码、多层协调、自适应 λ）全部否定——噪声只能破坏、不能精准纠正。

**AAC-lite**（`main_aac_lite.py`）首次取得了正向结果：计算"幻觉方向" `d = normalized(μ_incorrect - μ_correct)`，在推理时将隐藏状态沿该方向投影减去，能提升知识筛选集的准确率（+3.2pp）。

I1 是 AAC-lite 的系统深化：**用 PCA 和 LDA 替代简单的 mean-diff，检验是否存在更优的"对比方向"，并严格验证方向性前提。**

---

## 2. 核心原理

### 2.1 隐空间方向假设

假设在某一中间层 L 的残差流 `h ∈ R^d` 中，正确样本和错误样本的隐藏状态占据不同的区域：

```
h(correct)   ~ cluster centered at μ_c
h(incorrect) ~ cluster centered at μ_i
```

存在一个"幻觉方向" `d_halluc`（指向错误样本簇），使得：
- **subtract（减去该方向分量）**：将 h 拉向正确区域 → 准确率↑
- **add（加上该方向分量）**：将 h 推向错误区域 → 准确率↓

如果 subtract 和 add **同时提升**准确率（如早期噪声实验所见），则干预不是方向性的——只是在注入随机扰动。**方向性检查（subtract↑ + add↓）是判断方向是否有意义的必要条件。**

### 2.2 三种方向构造方法

#### (1) Mean-Diff（AAC-lite 基线）

```
d_mean_diff = normalized(μ_incorrect - μ_correct)
```

最朴素的方法：正确与错误样本均值的差向量。计算复杂度 O(n·d)，无超参数。

**局限**：样本量小（300）时，均值估计噪声大；且如果正确/错误的 cluster 有显著内部结构（如多个子类），均值差可能混入方向不纯的噪声。

#### (2) PCA 方向

```
r_i = h_i^incorrect - μ_correct          # error residuals from correct centroid
R = stack([r_1, ..., r_n_error])         # [n_error, d_model]
d_pca = PC1(R)                            # first principal component
```

只关注**错误样本偏离正确质心的残差**的 PC1。直觉：如果错误样本的偏差集中在某个低维子空间，PC1 能更纯粹地捕获这个方向。

#### (3) LDA 方向

经典 Fisher 线性判别：寻找能最大化类间方差 / 类内方差比的方向。

```
S_W = within-class covariance (Ledoit-Wolf shrunk)
w = (S_W + γ·trace(S_W)/d · I)^{-1} · (μ_i - μ_c)      # regularized LDA
d_lda = normalized(w)
```

**2018维 << 300样本** 时 S_W 严重秩亏（rank ≤ 299），直接求逆不稳定。I1 使用两步正则化：
1. Ledoit-Wolf 收缩估计 S_W（将样本协方差向单位矩阵收缩）
2. Ridge 惩罚（γ·trace(S_W)/d · I）确保满秩可逆

#### (4) Random（对照）

随机单位向量 `d_rand ~ N(0, I) / ||d_rand||`，验证方向效应不是任何扰动都能产生的统计噪声。

## 3. 实现细节

### 3.1 三阶段流程

```
Phase 1: 在训练集上收集隐藏状态
Phase 2: 计算 4 种方向（mean-diff / PCA / LDA / random）
Phase 3: 在验证集上扫描评估 [method × layer × λ × mode] 全组合
```

**代码路径**：`experiments/phase2_entropy/main_i1_directions.py`

### 3.2 Phase 1：收集隐藏状态

```python
# 加载 HellaSwag train split，随机取 n_dir=300 样本
ds_train = load_dataset("Rowan/hellaswag", split="train")
dir_samples = ds_train.shuffle(seed=42).select(range(300))

# 对每个样本做前向传播，通过 TransformerLens hooks 截取
# blocks.{L}.hook_resid_post 处的隐藏状态（最后一个 token 位置）
# 按模型预测是否正确分区存储
accum = {L: {"correct": [h_1, ...], "incorrect": [h_j, ...]}}
```

关键细节：
- 仅取**最后一个 token 位置**的残差流（即预测 token 前的状态）
- 分区依据是**模型自己的预测**是否正确（不是标签是否正确）——因为我们干预的是模型的内部表示，需要对齐模型"认为"的对错
- `format_prompt` 将 HellaSwag 的 ctx + 4 个选项拼接为统一格式

### 3.3 Phase 2：方向计算

```python
# mean-diff
d_md = normalized(mean(accum["incorrect"]) - mean(accum["correct"]))

# PCA
R = stack([h - μ_correct for h in accum["incorrect"]])  # error residuals
pca = PCA(n_components=1).fit(R)
d_pca = normalized(pca.components_[0])

# LDA
X_c, X_i = accum["correct"], accum["incorrect"]  # numpy float64
S_W = LedoitWolf().fit(concat(centered(X_c), centered(X_i))).covariance_
S_W_reg = S_W + γ · trace(S_W)/d · I
d_lda = normalized(solve(S_W_reg, μ_i - μ_c))

# Random
d_rand = normalized(randn(d_model))
```

### 3.4 Phase 3：干预评估

**最核心的创新——投影 Hook**：

```python
def make_projection_hook(direction, lam, mode):
    sign = -1.0 if mode == "subtract" else 1.0
    d = direction  # [d_model] unit-norm

    def hook(activation, hook=None):
        # activation: [batch, seq, d_model]
        proj_mag = activation @ d                    # [batch, seq] — 标量投影长度
        projection = proj_mag.unsqueeze(-1) * d       # [batch, seq, d_model] — 向量投影
        return activation + sign * lam * projection   # 沿方向加/减 λ·投影
    return hook
```

这不是简单地加噪声 `ε ~ N(0, σ²I)`，而是：
1. 计算每个 token 的隐藏状态 h 在方向 d 上的投影分量 `proj_d(h) = (h·d) · d`
2. 根据 mode 减去或加上 λ 倍的该分量
3. Hook 注册在 `blocks.{L}.hook_resid_post`，在残差流经过该层后执行

**干预配置**：2 层 (L11, L15) × 4 方法 (mean_diff, pca, lda, random) × 4 λ (0.1, 0.3, 0.5, 1.0) × 2 mode (subtract, add) = **64 配置**

**评估指标**：
- 全量准确率 Δ：`acc(干预后) - acc(基线)`
- 知识筛选准确率 Δf：仅统计 P(correct) > 0.3 的样本子集
- 方向性判定：`subtract Δf > 0` 且 `add Δf < 0`（或相反方向也行）→ directional = YES

## 4. 实验结果

### 4.1 方向一致性

运行日志显示 PCA 与 mean-diff 高度一致：

```
PCA cos with mean-diff = -0.88  (负号源于符号约定差异)
LDA cos with mean-diff = -0.02  (近似正交，高维小样本失败)
```

**PCA ≈ mean-diff**：错误样本残差近似各向同性（没有主导的误差方向），PC1 提取的变异方向恰好与均值差对齐。

**LDA 失败**：d_model=2048，n=300，S_W 有效秩 < 300。即便加权收缩+Ledoit-Wolf，LDA 仍无法学到有意义的类别判别方向。这是高维小样本的必然结果。

### 4.2 方向性检查（核心验证）

| Method | Layer | Subtract Δf | Add Δf | 方向性 |
|--------|-------|:-----------:|:------:|:------:|
| mean_diff | **L11** | **+5.50pp** | -1.35pp | ✓ YES |
| mean_diff | L15 | +1.93pp | +1.67pp | ✗ no |
| pca | **L11** | **+5.61pp** | -2.08pp | ✓ YES |
| pca | L15 | +0.58pp | +0.45pp | ✗ no |
| lda | L11 | +0.60pp | -0.58pp | ✓ YES |
| lda | L15 | -0.58pp | -3.97pp | ✗ no |

**结论**：
- **L11 通过方向性检查**（三种方法均 subtract↑ + add↓），L15 未通过（add 反而也提升或无明显方向）
- mean-diff 和 PCA 效果几乎相同（+5.5pp vs +5.6pp），PCA 无增量优势
- LDA L11 虽然方向性通过，但效应量极小（+0.6pp），不如 mean-diff

### 4.3 最优配置

| 方法 | 层 | λ | 模式 | Acc(全量) | Δ | Acc(过滤) | Δf |
|------|---|----|------|----------:|----:|----------:|----:|
| mean_diff | L11 | 0.5 | subtract | 58.5% | +3.5pp | **92.9%** | **+5.56pp** |
| pca | L11 | 1.0 | subtract | 59.0% | +4.0pp | 92.9% | +5.61pp |
| random | — | — | — | — | ~0 | — | ~0 |

- 最优 λ=0.5（不是更大的 1.0），说明过度投影也会损害有用信息
- Random 方向 Δf ≈ 0，确认有意义方向不是"任何方向扰动"的产物
- PCA 和 mean-diff 的表现差异在统计噪声范围内

## 5. 局限与矛盾

### 5.1 核心矛盾：I1 vs S1

I1 在 L11 用 λ=0.5 subtract 获得 **+5.56pp**，但在 S1（检测+干预全流程实验）中：

| 指标 | I1（方向训练集） | S1（独立 200 样本） |
|------|:----------------:|:-------------------:|
| Δ (filtered) | +5.56pp | **-1.67pp** |

S1 用不同的 200 样本加载 I1 计算的方向做干预，结果不仅没有提升，反而退化。

**可能原因**：
1. I1 的方向在 300 训练样本上**过拟合**——捕获的是样本特定的噪声方向，而非真正的"幻觉方向"
2. 200 样本太小，S1 仅 120 个知识筛选样本，方差大
3. L11 的方向投影效应本身就不稳健——300 样本不足以学到 2048 维空间中稳定可靠的分类方向

### 5.2 P0 待验证

当前 P0 计划：**500 样本 5-fold CV 重验证**，每 fold 用 400 样本算方向、100 样本评估。如果平均 Δ > 3pp，则方向确实有效，I1 vs S1 的矛盾是样本量/方差所致。如果平均 Δ ≈ 0，则"对比方向引导干预"整个思路需要重新审视。

## 6. 与前序实验的关系

```
AAC-lite (main_aac_lite.py)
  └─ 提出 mean-diff 方向 + 投影干预（首次正向结果 +3.2pp）
     │
I1 (main_i1_directions.py)  ← 本实验
  ├─ 增加 PCA 方向（检验 error residuals 是否有主导方差方向）
  ├─ 增加 LDA 方向（检验类别判别方向是否优于均值差）
  ├─ 增加 Random 对照（验证方向效应是否统计显著）
  └─ 增加方向性检查（subtract↑ + add↓ 必须同时成立）
     │
I2 (main_i2_multi_layer.py)
  └─ 多层联合投影（L11+L15），无协同增益，L11 only 始终最优
     │
I3 (main_i3_adaptive_lambda.py)
  └─ 自适应 λ = (1 - P(correct)) × λ_base，无实质提升
     │
S1 (main_s1_pipeline.py)
  └─ 检测+干预全流程，发现 I1 方向不跨样本迁移
```
